Politique de Confidentialité - Extension Cam'it !

Dernière mise à jour : 2024

1. Introduction

Cette politique de confidentialité décrit comment l'extension Chrome "Cam'it !" collecte, utilise et protège vos données personnelles.

2. Données Collectées

L'extension collecte et stocke les données suivantes :

2.1 Données d'Authentification

  • Token d'authentification : Stocké localement dans le navigateur (chrome.storage.local)
  • Utilisation : Permet de se connecter à l'application Cam'it ! et d'accéder à vos paniers

2.2 Données de Produits

  • Informations extraites : Nom, prix, image, lien URL du produit
  • Source : Extraites directement depuis la page web que vous visitez
  • Stockage : Temporairement dans la mémoire de l'extension, puis envoyées à l'application Cam'it !

2.3 Données de Navigation

  • URL de la page actuelle : Utilisée uniquement pour extraire les informations du produit
  • Aucune autre donnée de navigation n'est collectée

3. Stockage des Données

3.1 Stockage Local

  • Toutes les données sont stockées localement dans votre navigateur
  • Le token d'authentification est stocké dans chrome.storage.local
  • Aucune donnée n'est stockée sur des serveurs externes par l'extension

3.2 Transmission des Données

  • Les données de produits sont transmises uniquement à votre application Cam'it ! (backend)
  • La transmission se fait via des requêtes HTTPS sécurisées
  • Aucune donnée n'est transmise à des services tiers

4. Utilisation des Données

Les données collectées sont utilisées uniquement pour :

  1. Authentification : Se connecter à votre compte Cam'it !
  2. Extraction de produits : Extraire les informations depuis les pages web
  3. Ajout de produits : Ajouter des produits à vos paniers dans l'application

5. Partage des Données

  • Aucune donnée n'est partagée avec des tiers
  • Aucune donnée n'est vendue ou utilisée à des fins publicitaires
  • Les données sont utilisées uniquement pour le fonctionnement de l'extension

6. Sécurité

  • Toutes les communications sont chiffrées via HTTPS
  • Le token d'authentification est stocké de manière sécurisée dans le navigateur
  • Aucune donnée sensible n'est exposée dans le code de l'extension

7. Permissions de l'Extension

L'extension demande les permissions suivantes :

  • activeTab : Pour accéder à la page web actuelle et extraire les informations du produit
  • storage : Pour stocker le token d'authentification localement
  • scripting : Pour exécuter le script d'extraction sur la page
  • tabs : Pour obtenir l'URL de la page actuelle
  • host_permissions : Pour communiquer avec l'application Cam'it ! (localhost et serveur de production)

8. Vos Droits

Vous avez le droit de :

  • Supprimer vos données : Supprimez l'extension pour supprimer toutes les données stockées localement
  • Révoquer l'accès : Déconnectez-vous de l'application pour révoquer l'accès au token
  • Demander des informations : Contactez-nous pour toute question sur vos données

9. Cookies et Technologies Similaires

L'extension n'utilise pas de cookies ni de technologies de suivi.

10. Modifications de cette Politique

Nous nous réservons le droit de modifier cette politique de confidentialité. Les modifications seront publiées sur cette page avec une date de mise à jour.
